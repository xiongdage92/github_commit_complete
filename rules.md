# General Rules / 通用规则

- No hardcoded secrets（禁止硬编码密钥）
- Must include README（必须包含 README）
- Must include .gitignore（必须包含 .gitignore）
- Should include LICENSE（建议包含 LICENSE）

---

# Python

- Follow PEP8（遵循 PEP8 规范）
- Use docstrings（使用函数注释）

---

# JavaScript / TypeScript

- Prefer const/let over var（优先使用 const/let 而非 var）
- Avoid console.log in production（生产环境中避免使用 console.log）

---

# GitHub Best Practices / GitHub 规范

- Clear repo structure（结构清晰）
- No large files >50MB（禁止大于 50MB 的大文件）
- Meaningful documentation（文档完整且有意义）
